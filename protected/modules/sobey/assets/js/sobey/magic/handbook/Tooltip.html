气泡提示

飘浮在页面主体内容上的层，多用于提醒和提示。气泡提示具有一个箭头，可以设置箭头样式和位置。

参数配置

  • options.autoHide : Boolean
    是否自动隐藏, 如果为true,则会在scroll,resize,click,keydown(escape)事件下隐藏，默认值为true。
  • options.hasCloseBtn : Boolean
    是否可以通过右上角X来关闭提示。默认值为true。
  • options.hasArrow : Boolean
    是否显示箭头。默认值为true。
  • options.offsetX : Number
    定位时的偏移量，X方向。
  • options.offsetY : Number
    定位时的偏移量，Y方向。
  • options.target : Array
    需要提示的节点。(必选)
  • options.content : Function|String
    自定义内容定制。若为Function,则参数为Tangram对象(目标节点)。默认值为空。
  • options.showEvent : String
    提示显示的动作，默认值为mouseover,focus。
  • options.hideEvent : String
    提示隐藏的动作，默认值为mouseout,blur。
  • options.position : String
    设置tooltip的位置，值为top|bottom|left|right，默认值为bottom。
  • options.arrowPosition : Number|Percent
    设置arrow的位置，如果是上、下方位的话，都相对于左边的距离。如果是左、右方位的话，都相对于上面的距离。如果该值不存在，则自动计算提示框位置离目标节点中间最靠近的位置。

组件属性

组件方法

  • render

    渲染提示框

    • 参数

      • el: {HTMLElement|id|dom}

        渲染到目标容器下，如果该值存在，则提示框组件在el下，如果不存在且目标节点存在的情况下，提示框组件和目标节点是兄弟关系，如果都不存在，则渲染到body下。

  • reposition

    重新定位提示框位置

  • show

    显示提示框

  • hide

    隐藏提示框

  • setTarget

    设置目标节点

  • setContent

    设置提示框内容

  • setPosition

    设置提示框位置

组件事件

  • onload

    提示框渲染完时触发

  • onbeforeshow

    显示提示框之前触发,如果返回为false,则阻止显示。

  • onshow

    显示提示框时触发

  • onbeforehide

    隐藏提示框之前触发,如果为false,则阻止隐藏。

  • onhide

    隐藏提示框时触发

  • ontargetchange

    当目标节点发生变化时触发。
